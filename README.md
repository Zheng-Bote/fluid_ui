# Fluid UI CSS3 Framework

Fluid UI is a lightweight, modern CSS framework built for performance and flexibility. It leverages the power of modern CSS features to provide a robust and visually appealing user interface.

## Key Features

- **OKLCH Colors**: Utilizes the OKLCH color space for better perceptional consistency and vibrant color palettes.
- **Fluid Typography**: Implements `clamp()` based typography to ensure text scales smoothly across all device sizes.
- **Modular Layout**: A flexible layout system including sidebar configurations and responsive grids.
- **Rich Components**: Includes a comprehensive set of styled components:
  - Buttons (Primary, Outline, Ghost, Icon)
  - Inputs (Text, Checkbox, Radio, Toggle switches)
  - Cards (Standard, Status variants)
  - Message Boxes (Success, Error, Info, Warning)
  - Navigation (Breadcrumbs, Pagination, Tabs)
  - Modals, Galleries, Accordions, and Toasts.
- **Dark/Light Mode**: Built-in support for theme switching.

## Usage

This framework is designed for use with the gh_docs_bot Static Site Generator, but can be easily adapted to your own needs by modifying the index.html.
